Bongard Test

I learned about Bongard problem reading a blog post of David Chapman titled: A first lesson in meta-rationality | Meta-rationality

It inspired me to create a test based on the Bongard problem.

Bongard problem was created by Russian computer scientist Mikhail Moiseevich Bongard (Bongard problem - Wikiwand).

Bongard problems can be culture fair or not and offer a vast creative level of freedom.

There is an index of Bongard problems maintain by Harry Foundalis (he wrote a dissertation on the topic).

Currently there is 2 tests available.
